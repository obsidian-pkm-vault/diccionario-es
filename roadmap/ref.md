## Proyectos existentes

| Project                                                           | Type                     | Language(s)                     | What it offers                         |
|-------------------------------------------------------------------|--------------------------|---------------------------------|----------------------------------------|
| [Wiktionary](https://www.wiktionary.org/)                         | Diccionario colaborativo | Multilenguaje                   | Definiciones, etimologías y categorías |
| [RAE / DLE](https://dle.rae.es/)                                  | Diccionario oficial      | Español                         | Referencia principal del español       |
| [rae-api.com](https://rae-api.com/)                               | Diccionario RAE (API)    | Español                         | API moderna para DLE                   |
| [WordReference](https://www.wordreference.com/)                   | Diccionario/traducción   | Multilenguaje                   | Relaciones léxicas y traducciones      |
| [Etimologías de Chile](https://etimologias.dechile.net/)          | Etimología               | Español                         | Raíces y evolución lingüística         |
| [Wikidata](https://www.wikidata.org/wiki/Wikidata:Main_Page)      | Knowledge graph          | Multilenguaje                   | Entidades y relaciones estructuradas   |
| [FrameNet](https://framenet.icsi.berkeley.edu/frames)             | NLP semántico            | Inglés                          | Frames y relaciones conceptuales       |
| [BabelNet](https://babelnet.org/)                                 | Ontología/grafo          | Multilenguaje                   | Une WordNet, Wikipedia y Wikidata      |
| [GlobalWordNet Schemas](https://globalwordnet.github.io/schemas/) | Schema/estándar          | Multilenguaje                   | Estándar para definir WordNets         |
| [ConceptNet](https://conceptnet.io/)                              | Semantic graph           | Multilenguaje                   | Relaciones semánticas y commonsense    |
| [OMW](https://open-multingual-wordnet.github.io/)                 | Semantic graph           | Multilenguaje (incluye español) | WordNets sincronizados entre idiomas   |
| [WordNet](https://en-word.net/)                                   | Semantic graph           | Inglés                          | Relaciones léxicas y semánticas        |
| [Curated Wordnets](https://github.com/omwn/curated-wordnets)      | Semantic graph (dataset) | Multilenguaje                   | WordNets curados y alineados (OMW)     |
| [Cygnet](https://cygnet.maudslay.eu/)                             | Semantic explorer        | Inglés (OMW-based)              | Visualización navegable de OMW         |
| [Lexipedia](https://www.lexipedia.com/)                           | Semantic explorer        | Inglés                          | Visualización navegable tipo WordNet   |
